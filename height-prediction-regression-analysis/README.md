# Predicting Height with Linear Regression

Statistical analysis project focused on predicting students' height using hand span measurements through simple linear regression.

## Overview

This project applies a complete linear regression workflow using real data from 167 university students. Beyond fitting the regression model, the analysis includes diagnostic validation, hypothesis testing, confidence intervals, prediction intervals, and lack-of-fit assessment.

## Objectives

- Model the relationship between hand span and height
- Evaluate the significance of the linear relationship
- Validate regression assumptions through residual analysis
- Compare confidence and prediction intervals
- Interpret results in a practical predictive context

## Techniques Used

- Simple Linear Regression
- ANOVA for regression
- Residual diagnostics
- Shapiro-Wilk normality test
- Breusch-Pagan homoscedasticity test
- Durbin-Watson independence test
- Lack-of-fit test
- Confidence and prediction intervals

## Technologies

- R
- tidyverse
- ggplot2
- knitr
- kableExtra
- ggfortify
- lmtest

## Key Results

- Strong positive linear relationship between hand span and height
- Regression assumptions were satisfied
- No evidence of lack of fit
- Statistically significant regression coefficient
